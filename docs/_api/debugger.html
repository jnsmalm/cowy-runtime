Debugging

Gameplay has a debugging tool built-in. To start the debugger you pass the 'debug' argument when starting the play program.

% play debug script.js

To set a breakpoint, place the debugger; statement where you want the debugger to stop.

var sum = 0;
function add(value) {
  sum += value;
}
for (var i=0; i<5; i++) {
  add(i);
  debugger;
}

The debugger stops and displays the source code at the breakpoint. The debug prompt appears and waits for your input.

break at script.js:7
  5 for (var i=0; i<5; i++) {
  6   add(i);
> 7   debugger;
  8 }
debug>

The following commands are available at the debug prompt:

  • cont - Continue to the next breakpoint
  • next - Step to the next statement
  • in - Step into the next statement
  • out - Step out from the current block

You can also press 'enter' to execute the previous debug command. If you want to inspect a value for a variable, just type the name of the variable. You can also enter any other JavaScript statement.

debug> sum
6
debug> 1+2
3
