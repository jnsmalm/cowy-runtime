Texture2D

Loads an image from file, used when drawing primitives. Supported image files: jpg, png, tga, bmp, psd, gif, hdr, pic. A window must exist before creating a texture, otherwise an exception will be thrown.

var texture = new Texture2D('image.png');
 
// Gets the width/height of the texture.
var w = texture.width;
var h = texture.height;

Color data

With the function getData you can get the color data for that texture. The result is an array with all the colors. If you want to get a color at a specific pixel you must take into consideration the channels property. channels can have a value of 1, 2, 3 or 4. E.g. the value of channels will be 3 when the image only has RGB channels (without alpha).

// var texture = new Texture2D(...);
var colors = texture.getData();

function getPixelRGBA(x, y) {
  var i = y * texture.width * texture.channels + x * texture.channels;
  return {
    r: colors[i],
    g: colors[i + 1],
    b: colors[i + 2],
    a: colors[i + 3],
  };
}

// Get pixel at x = 100, y = 25
var pixelColor = getPixelRGBA(100, 25);
