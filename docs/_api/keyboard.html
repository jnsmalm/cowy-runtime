Keyboard

Handles input state from a keyboard. Both isKeyDown and isKeyPress functions takes a key code as an argument. Key codes can be found at the GLFW website.

// var window = new Window(...)
var keyboard = new Keyboard(window);
 
// Update the state for the keyboard.
keyboard.updateState();
 
if (keyboard.isKeyDown('A'.charCodeAt(0))) {
  // The 'A' key is down.
};
if (keyboard.isKeyPress(32)) {
  // The space key was just pressed.
};
