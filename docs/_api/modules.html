Modules

It wouldn't be great if you had to code the whole game in a single file. A module is a part of a program used to encapsulate logic and functionality. The global module object is used to export only the functionality you want to expose to other files. All variables and functions not exported are by default local to the current file.

The contents of foo.js

var score = require('/score.js');
score.destroyedMinorShip();
score.destroyedLargeShip();
score.print();

The contents of score.js

var score = 0;
 
module.exports.destroyedMinorShip = function () {
  score += 10;
};
 
module.exports.destroyedLargeShip = function () {
  score += 50;
};

module.exports.print = function () {
  console.log('score: ' + score);
}
