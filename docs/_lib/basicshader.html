BasicShader

Predefined vertex/fragment-shader that supports vertices that have a position, normal and texture.

var BasicShader = require('lib/shader.js').BasicShader;

// var window = new Window(...)
var basicShader = new BasicShader(window);
 
// var texture = new Texture2D(...)
basicShader.setMaterialTexture(texture);
 
// Set the shader program for the graphics.
window.graphics.setShaderProgram(basicShader.shaderProgram);

Vertex

The vertex used with BasicShader has support for position, normal and texture coordinates. When creating the vertex, none of these properties are required (they all have default values). You will want to at least set the position though.

var vertex = new BasicShader.Vertex({
    position: vec3.fromValues(0,1,0),
    normal: vec3.fromValues(0,1,0),
    textureCoords: vec2.fromValues(0.5,0)
});

When setting the vertex declaration for the VertexDataState, the declaration from the BasicShader vertex can be used.

// var vertexDataState = new VertexDataState(...);
vertexDataState.setVertexDeclaration(BasicShader.Vertex.declaration(), basicShader.shaderProgram);

VertexList

Holds a list of vertices. To set vertex data for a VertexDataState, the toArray function can be used.

// var vertex1 = new BasicShader.Vertex(...)
// var vertex2 = new BasicShader.Vertex(...)
// var vertex3 = new BasicShader.Vertex(...)
 
// Create the vertex list with the specified vertices.
var vertices = new BasicShader.VertexList([ vertex1, vertex2 ]);
 
// Vertices can also be added with a function.
vertices.add(vertex3);

// var vertexDataState = new VertexDataState(...)
vertexDataState.setVertices(vertices.toArray());

Material

The material (appearence) for the drawn objects using BasicShader can be changed by a number of different functions.

// var texture = new Texture2D(...);

basicShader.setMaterialTexture(texture);
basicShader.setMaterialColor(vec3.fromValues(1,0,0));
basicShader.setMaterialSpecularColor(vec3.fromValues(1,1,1));
basicShader.setMaterialShininess(0.5);
