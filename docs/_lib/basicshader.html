BasicShader

A basic predefined vertex/fragment-shader that supports vertices that have a position, color and texture.

var BasicShader = require('lib/shader.js').BasicShader;

// var window = new Window(...)
var basic = new BasicShader(window);
 
// var texture = new Texture2D(...)
basic.setTexture(texture);
 
// Set the shader program for the graphics.
window.graphics.setShaderProgram(basic.shaderProgram);

Vertex

The vertex used with BasicShader has support for position, color and texture coordinates. When creating the vertex, non of these properties are required (they all have default values). You will want to at least set the position though. When creating the VertexBuffer, the vertex declaration from the vertex is used.

var vertex1 = new BasicShader.Vertex({
    position: { x: 0, y: 1, z: 0 },
    color: { r: 1, g: 1, b: 1, a: 1 },
    textureCoordinates: { u: 0.5, v: 0 }
});
var vertex2 = new BasicShader.Vertex({
    position: { x: -1, y: -1, z: 0 },
    color: { r: 1, g: 1, b: 1, a: 1 },
    textureCoordinates: { u: 1, v: 1 }
});
var vertex3 = new BasicShader.Vertex({
    position: { x: 1, y: -1, z: 0 },
    color: { r: 1, g: 1, b: 1, a: 1 },
    textureCoordinates: { u: 0, v: 1 }
});

// var window = new Window(...)
var vertexBuffer = new VertexBuffer(
    window.graphics, BasicShader.Vertex.declaration());

VertexList

Holds a list of vertices. To set data for a vertex buffer the toArray function can be used.

// var vertex1 = new BasicShader.Vertex(...)
// var vertex2 = new BasicShader.Vertex(...)
// var vertex3 = new BasicShader.Vertex(...)
 
// Create the vertex list with the specified vertices.
var vertices = new BasicShader.VertexList([ vertex1, vertex2 ]);
 
// Vertices can also be added with a function.
vertices.add(vertex3);

// var vertexBuffer = new VertexBuffer(...)
vertexBuffer.setData(vertices.toArray());
