{ Gameplay }

Platform for creating desktop games with JavaScript. Easy to distribute and requires no installation. Supported on Mac OS X and Windows. It's free and open source.

Download v0.3.0 for Mac OS X

The download includes runtime, library and examples.

Download v0.3.0 for Windows

The download includes runtime, library and examples.

Overview

Gameplay executes JavaScript using Google's V8 JavaScript Engine and an API to draw graphics, handle input and play sounds is built-in. Gameplay does not run inside a web browser, instead it runs as a native application on the current platform. The runtime is a single executable file, easy to distribute so your end users won't have to install anything on their desktop computer for playing your game.

Getting started

Download and unzip the files wherever you choose. Execute an included example with the play runtime from the command line. There are more examples included, browse through them to learn more.

% play examples/mesh.js
